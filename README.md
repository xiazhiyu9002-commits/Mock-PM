# 产品经理模拟 - 个人练习仓库 📘

> 用于记录我个人在产品经理技能上的模拟练习：PRD、竞品分析、思维导图

本仓库是我日常打磨产品经理核心能力的“沙盒”。练习于撰写产品需求文档（PRD）、开展竞品分析、绘制思维导图，并通过持续记录来复盘自己的逻辑与决策。

## 🎯 使用目的

- 📄 **模拟PRD撰写**：练习不同产品形态（移动端/后台/小程序等）的PRD，包括需求背景、功能清单、流程图、异常说明等。
- 🔍 **模拟竞品分析**：选取主流产品进行结构化对比（SWOT、KANO、用户体验五要素等），输出分析报告。
- 🧠 **模拟思维导图**：用导图梳理产品架构、用户旅程、优先级排序，强化结构化思维。

## 📁 目录结构

- 每一个包都有PRD、竞品分析、思维导图模拟文件
